Help (Guidelines for New Staff)

5. Help Section (Guidelines for New Staff)

System Purpose
- This Reception Dashboard is used to:
  • Add new room reservations
  • View and manage reservations list
  • Calculate bills and print invoices
  • Read help instructions
  • Logout safely when work is finished


A) Login & Safety
1) Enter your username and password on the Login page.
2) Do not share your password with anyone.
3) Always click Logout when you finish your shift.


B) Add Reservation (Add Reservation Tab)
Steps:
1) Click Add Reservation tab.
2) Click New if you want a new reservation number.
3) Select Room Type.
4) Select Check-in date and Check-out date.
5) Enter Guest Name.
6) Enter Contact No (required).
7) Enter Address (optional).
8) Click Save Reservation.

Notes:
- Reservation number is auto generated.
- If the contact number already exists, the system may reuse the guest details.


C) View Reservations (Reservations List Tab)
How to use filters:
- Status
  • Active = currently staying
  • Upcoming = future reservations
  • Completed = check-out date is before today
- Room Type = filter by selected room type
- Search = type reservation number / guest name / contact number

Tips:
- Click Refresh to load latest list.
- Always double check dates before confirming a booking.


D) Bill & Print (Bill & Print Tab)
Steps:
1) Open Bill & Print tab.
2) Enter Reservation No (example: R20260225-0001).
3) Click Get Bill.
4) Check the bill details.
5) Click Print to print the invoice.

Important:
- Print button will be enabled only after bill is loaded.

F) Exit / Logout (Exit Tab)
Steps:
1) Click Exit tab.
2) Click Logout.
3) Close the browser tab after logout.


Common Troubleshooting
- Can’t find a reservation: Go to list tab and search by contact number.
- Bill not showing: Check reservation number format and try again.
- Wrong dates: Always confirm check-in/check-out before saving.

Exit System

Use Logout then close the tab.
