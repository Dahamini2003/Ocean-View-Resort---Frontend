Help

New Admin Guide

Admin Quick Start

  1. Login with your ADMIN account.
  2. Check Dashboard for quick reservation counts.
  3. Use Add Reservation to create new bookings.
  4. Use Update/Delete Reservations to edit or remove bookings.
  5. Use Guest Details to update guest name, contact, and address.

Daily Admin Tasks

  • Room Types (CRUD): Add / update room type and rate per night.
  • Users (CRUD): Create receptionist accounts, change roles, reset passwords.
  • Search: Use reservation number / guest / contact search to find fast.
  • Refresh: Press Refresh buttons after changes.

Step-by-step: Add a Reservation

  1. Open Add Reservation.
  2. Click New to generate a reservation number automatically.
  3. Select Room Type.
  4. Select Check-in and Check-out dates.
  5. Enter Guest Name, Contact No and Address.
  6. Click Save Reservation.
  7. Go to Update/Delete Reservations to confirm it is listed.

Troubleshooting

  • Reservation not visible? Click Refresh and check filters like Status and Room Type.
  • Cannot update guest? Load guest using contact number first, then click Update Guest.
  • Room type missing? Add it in Room Types (CRUD) and refresh the page section.
  • Login issue? Reset the password from Users (CRUD).
